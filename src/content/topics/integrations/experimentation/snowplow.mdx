## Overview

This topic links to Snowplow's documentation for its LaunchDarkly integration. Snowplow is a customer data collection and behavioral analytics tool. You can use Snowplow with LaunchDarkly's Experimentation feature to forward events in Google Tag Manager to LaunchDarkly. You must be using LaunchDarkly's Experimentation product to integrate with Snowplow. To learn more, read [About Experimentation](/home/about-experimentation).

Snowplow manages this integration. To learn how to use it, read Snowplow's [LaunchDarkly Tag for GTM SS](https://docs.snowplow.io/docs/destinations/forwarding-events/google-tag-manager-server-side/launchdarkly-tag-for-gtm-ss/).
