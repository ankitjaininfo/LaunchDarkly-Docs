## Overview

This topic links to Census' documentation for its LaunchDarkly integration. Census is a data activation tool you can use to send metric events from any supported data source to LaunchDarkly. You must be using LaunchDarkly's Experimentation product to integrate with Census. To learn more, read [About Experimentation](/home/about-experimentation).

Census manages this integration. To learn how to use it, read Census' [LaunchDarkly documentation](https://docs.getcensus.com/destinations/launchdarkly).
