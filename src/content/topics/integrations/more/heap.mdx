## Overview

This topic links to Heap's documentation for its LaunchDarkly integration. Heap is a customer behavioral data analytics tool. You can send feature flags and context attributes to Heap as event and user properties.

Heap manages this integration. To learn more, read Heap's [LaunchDarkly Integration documentation](https://help.heap.io/integrations/testing-and-personalization/launchdarkly/).
