## Overview

The CloudQuery LaunchDarkly plugin pulls data from LaunchDarkly and loads it into any supported CloudQuery destination.

CloudQuery manages this integration. To learn how to use it, read CloudQuery's [LaunchDarkly Source Plugin](https://www.cloudquery.io/docs/plugins/sources/launchdarkly/overview).
