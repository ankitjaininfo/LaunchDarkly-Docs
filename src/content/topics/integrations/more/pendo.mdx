## Overview

This topic links to Pendo's documentation for its LaunchDarkly integration. Pendo is a user insights, feedback, and engagement tool. You can access Pendo audiences with LaunchDarkly targeting rules to deliver new features to the right contexts.

To learn more about how Pendo and LaunchDarkly work together, read [Launched: Pendo Integration with LaunchDarkly](https://launchdarkly.com/blog/launched-pendo-integration-with-launchdarkly/).

Pendo manages this integration. To learn more, read Pendo's [LaunchDarkly Recipe](https://support.pendo.io/hc/en-us/articles/360032592312-LaunchDarkly-Recipe-Automate-the-feature-release-process).
