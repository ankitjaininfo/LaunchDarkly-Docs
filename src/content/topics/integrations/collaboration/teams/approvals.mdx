## Overview

This topic explains how you can receive approval notifications in Microsoft Teams. Approvals let you request review for changes you plan to make to a flag. You can view approval requests from within the LaunchDarkly Microsoft Teams app.

To use this feature, you must have installed and configured the LaunchDarkly Microsoft Teams app. To learn how, read [Setting up the Microsoft Teams app](/integrations/microsoft-teams/setting-up).

To learn more about approvals, read [Approvals](/home/feature-workflows/approvals).

## Viewing approvals in Microsoft Teams

Here's how to view approval requests in the Microsoft Teams app:

1. Click the approval notification from the LaunchDarkly app.
2. Click **Open in LaunchDarkly** to view the flag details in the LaunchDarkly user interface.

   ![A Microsoft Teams notification from the LaunchDarkly app.](teams-ld-approval-notification.png)

Any reviewers that you added to the approval request also receive a Microsoft Teams notification if they have authorized the LaunchDarkly app.
