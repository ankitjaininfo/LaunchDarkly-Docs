## Featured integrations

Here are some of our most popular integrations:

- Using the **Slack integration**, you can view and filter flags and resources, subscribe to flag notifications, toggle flag targeting on and off, and more. To learn more about the integration, read [Slack](/integrations/slack).
- Using the **Jira** integration, you can link your feature flags to your team's Jira issues. To learn more about this integration, read [Jira Cloud](/integrations/jira).
- Using the **Datadog** integration, you can set up Datadog Events to receive any activity from LaunchDarkly. To learn more about this integration, read [Datadog](/integrations/datadog).

To learn about common problems integrations can help solve, read [Integrations use cases](/guides/integrations/integrations-use-cases).

## Building your own integration

LaunchDarkly's integration framework lets you use a standard configuration to build two-way integrations from LaunchDarkly to third-party services. To learn how to create your own integration, read [Using the LaunchDarkly integration framework](/integrations/building-integrations).
