## Overview

This topic explains the different event kinds for Snowflake Data Export destinations. The JSON schemas for Snowflake are formatted differently than LaunchDarkly's.

To learn more about the Snowflake destination, read [Snowflake's documentation](https://docs.snowflake.com/).

Snowflake Data Export destinations only support [`feature` events](#feature-events). A feature event is emitted when a feature flag connected to an experiment is evaluated.

Snowflake events include additional information about different LaunchDarkly properties. These properties are included in all event kinds.
